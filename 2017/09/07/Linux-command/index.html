Linux常用命令小结

Posted on 2017-09-07   |   In linux   |     |   visitors
  • cat 查看文件内容
    举例：查看文件名为test.txt的内容

  • iostat 检测系统io状态（一般指磁盘的io情况）

  • netstat 查看端口占用情况

  • ps 用来显示当前运行的进程（经常结合grep使用）

  • grep 过滤打印匹配的行
    举例：显示当前运行的Java进程

  • top 查看CPU当前的负载

  • touch 创建文件
    举例：创建文件名为test.txt的文件

  • chmod 修改文件权限

备注：777,表示权限的参数代码，一个文件的最高权限应该就是这个了，可以认为是所有用户都可以修改、读、执行这个文件
